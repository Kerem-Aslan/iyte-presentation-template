# EN: Unofficial Latex Beamer Presentation Template for Izmir Institute of Technology (IZTECH)

An unofficial LaTeX Beamer–based presentation template prepared for my graduation thesis.
The color palette and layout have been adjusted to be compatible with IZTECH’s colors.
You may modify it according to your needs and preferences.

### Acknowledgment

This template is based on the Beamer Presentation template provided by LatexTemplates.com:

- https://www.latextemplates.com/template/beamer-presentation

The original template has been customized and adapted for IZTECH presentations.

# TR: İzmir Yüksek Teknoloji Enstitüsü (İYTE) için Resmi olmayan Latex Beamer Sunum Şablonu

Bitirme tezim için hazırladığım Latex Beamer tabanlı resmi olmayan sunum şablonu.
Renk paleti ve düzen İYTE'nin renkleri ile uyumlu olacak şekilde ayarlandı.
İhtiyacınıza ve zevklerinize göre değiştirebilirsiniz.

Bu şablon, LatexTemplates.com üzerinde yer alan Beamer Presentation şablonu baz alınarak oluşturulmuştur:

- https://www.latextemplates.com/template/beamer-presentation

Orijinal şablon, İYTE sunumlarına uygun olacak şekilde düzenlenmiştir.
